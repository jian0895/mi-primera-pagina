implementaremos la libreria de turtle desde cero

copiamoa el codigo inicial

```python
import turtle

t = turtle   # Crea una tortuga
t.avanzar(100)        # Avanza 100 unidades
turtle.done()         # Mantiene la ventana abierta
```

se crean las funciones para que la tortuga se mueva.

```python
avanzar_arriba()      # **Mover hacia arriba**
avanzar_derecha()     # **Mover hacia la derecha**
avanzar_abajo()       # **Mover hacia abajo**
avanzar_izquierda()   # **Mover hacia la izquierda**
```
